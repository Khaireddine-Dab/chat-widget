Add this code before the closing </body> tag of your website:

<!-- Chat Widget -->
<div id="chat-widget-root"></div>
<script>
  window.chatWidgetConfig = {
    brandName: "Your Business",
    brandColor: "#8b5cf6",
    brandLogo: "/logo.png",
    welcomeMessage: "Hi! How can we help you today?",
    position: "bottom-right",
    businessPhone: "+1234567890",
    language: "en",
    quickReplies: [
      "I have a question",
      "I need support",
      "Book a reservation"
    ],
    enableBooking: true,
    enableEscalation: true,
    enableOfflineMode: true,
    businessHours: {
      monday: "9:00 AM - 6:00 PM",
      tuesday: "9:00 AM - 6:00 PM",
      wednesday: "9:00 AM - 6:00 PM",
      thursday: "9:00 AM - 6:00 PM",
      friday: "9:00 AM - 6:00 PM",
      saturday: "10:00 AM - 4:00 PM",
      sunday: "Closed"
    }
  };
</script>
<script src="https://your-domain.com/chat-widget.js"></script>

‚öôÔ∏è Step 2: Customize the configuration

Edit the window.chatWidgetConfig object to customize your widget:

brandName - Your business name (e.g., "Acme Corp")
brandColor - Primary color in hex format (e.g., "#8b5cf6")
brandLogo - URL to your logo image
welcomeMessage - First message users see
position - "bottom-right" or "bottom-left"
businessPhone - Your phone number for call feature
language - "en", "es", "fr", or "de"
quickReplies - Array of quick reply button texts
enableBooking - true/false to enable reservation form
enableEscalation - true/false to enable human agent escalation
enableOfflineMode - true/false to show offline message
businessHours - Object with hours for each day
